## 2. 単元名

教科書の単元名または題材名を記載。
サブタイトルをつける場合もある。

```markdown
## 単元名

「分数のたし算とひき算」
〜分母が異なる分数の計算〜
```

## 3. 単元の目標

**必ず3観点で記述する。**

学習指導要領の目標に準拠し、この単元で育成を目指す資質・能力を明確にする。

```markdown
## 単元の目標

### 知識及び技能
- 異分母分数の加法及び減法の意味について理解している。
- 異分母分数の加法及び減法の計算ができる。

### 思考力・判断力・表現力等
- 分数の意味や表現を基に、異分母分数の加法及び減法の計算の仕方を考え、説明している。

### 学びに向かう力・人間性等
- 異分母分数の加法及び減法の計算について、数学的に表現・処理したことを振り返り、多面的に捉え検討してよりよいものを求めて粘り強く考えようとしている。
```

## 5. 単元設定の理由

3つの観点から、なぜこの単元をこのように指導するのかを説明する。

### 児童観（生徒観）
- クラスの実態
- 既習事項の定着状況
- 興味・関心の傾向
- 学習上の課題

### 教材観
- この単元の教育的価値
- 教科全体における位置づけ
- 既習事項との関連
- 今後の学習への発展

### 指導観
- 具体的な指導の方針
- 「主体的・対話的で深い学び」の実現に向けた工夫
- 個に応じた指導の工夫
- ICT活用等の手立て

```markdown
## 単元設定の理由

### 児童観
本学級の児童は、○○について△△の傾向がある。事前調査では□□と回答した児童が○%であり、〜〜の課題が見られる。一方で、△△に対する関心は高く、〜〜

### 教材観
本単元は、学習指導要領第○学年の内容「A 数と計算」の（○）に位置付けられている。○○を理解することは、今後の△△の学習の基盤となり、〜〜

### 指導観
以上の児童観と教材観を踏まえ、本単元では以下の点を重視して指導する。
1. 〜〜
2. 〜〜
3. 〜〜
```

## 7. 本時の指導

## 8. 本時の展開（詳細版）

45分（小学校）または50分（中学校・高等学校）の授業の流れ。
**主発問・指示、予想される児童の反応、支援の手立てを必ず含める。**

### 本時の展開で必ず含める項目

| 項目 | 内容 |
|------|------|
| 時間配分 | 導入・展開・まとめの時間を明記 |
| 学習活動 | 児童生徒の活動を具体的に（番号付き） |
| 主発問・指示 | 教師の発問・指示を「」で具体的に記述 |
| 予想される児童の反応 | 複数パターンの反応を想定 |
| 指導上の留意点・支援 | つまずきへの手立て、机間指導の観点 |
| 評価 | どの観点で評価するか明記 |

### 詳細な記述例

```markdown
## 本時の展開

| 時間 | 学習活動 | 主発問・指示 | 予想される児童の反応 | 指導上の留意点・支援 | 評価 |
|------|---------|-------------|-------------------|-------------------|------|
| **導入** 5分 | 1. 前時の振り返り<br>・同分母分数の加法を確認 | 「前回はどんな計算をしましたか？」<br>「分母が同じ時、どうやって計算しましたか？」 | ・「分子をたした」<br>・「分母はそのまま」<br>・「2/5+1/5=3/5」 | ◯ 同分母の計算方法を確認し、本時との違いを意識させる<br>◯ 板書で計算方法を視覚化 | |
| | 2. 課題把握<br>・本時の課題を確認 | 「今日は 1/2+1/3 を考えます」<br>「前と何が違いますか？」 | ・「分母が違う」<br>・「どうやって計算するの？」<br>・「難しそう」 | ◯ 分母が異なることに着目させる<br>◯ 「どうすればできそう？」と見通しを持たせる | |
| **展開①** 15分 | 3. 自力解決<br>・個人で考える<br>・ノートに図や式を書く | 「まず自分で考えてみましょう」<br>「図や数直線を使ってもいいですよ」<br>「なぜその答えになるか、説明できるようにしましょう」 | ・数直線を描いて考える<br>・「1/2は3/6と同じ」と気づく<br>・「分母をそろえればいい」<br>・手が止まっている | ◯ 机間指導で多様な考え方を把握<br>◯ 手が止まっている児童には「1/2を他の分数で表せる？」とヒント<br>◯ 早く終わった児童には別の方法を考えさせる | |
| **展開②** 10分 | 4. グループ交流<br>・自分の考えを説明<br>・友達の考えを聞く<br>・質問し合う | 「グループで考えを伝え合いましょう」<br>「なぜそう考えたか、理由も説明してね」<br>「友達の考えで『なるほど』と思ったことは？」 | ・「私は通分で考えました」<br>・「〇〇さんと同じ」<br>・「数直線で考えた方がわかりやすい」<br>・「なぜ6が共通の分母なの？」 | ◯ 全員が発言できるよう声かけ<br>◯ よい説明の仕方を取り上げて全体に紹介<br>◯ 「なぜ？」「どうして？」の質問を促す | |
| **展開③** 10分 | 5. 全体共有<br>・代表が発表<br>・考え方を比較<br>・まとめる | 「〇〇さん、発表してください」<br>「他の考え方の人は？」<br>「どの方法でも共通していることは？」<br>「分母をそろえることを何といいますか？」 | ・「通分すればできる」<br>・「6にそろえた」<br>・「どっちも分母を同じにした」 | ◯ 複数の考え方を板書で整理<br>◯ 共通点（通分）を明確化<br>◯ 「通分」の用語を確認<br>◆ 計算の仕方を説明できているか | 思① |
| **まとめ** 5分 | 6. まとめ<br>・学習内容を整理 | 「今日の学習で分かったことをまとめましょう」 | ・「分母が違う分数のたし算は通分する」<br>・「分母をそろえてから計算する」 | ◯ キーワード（通分、分母をそろえる）を確認<br>◯ 児童の言葉でまとめを作る | |
| | 7. 振り返り<br>・振り返りカード記入 | 「振り返りを書きましょう」<br>「①わかったこと ②友達の考えで参考になったこと ③次にやってみたいこと」 | ・「通分の意味がわかった」<br>・「〇〇さんの数直線がわかりやすかった」<br>・「ひき算もやってみたい」 | ◯ 振り返りの視点を明示<br>◯ 次時への意欲につなげる | |
```

## 記述のポイント

### 学習活動の書き方
- 主語は児童・生徒
- 「〜する」の形で記述
- 具体的な活動内容を記載

### 指導上の留意点の書き方
- 主語は教師
- 「〜させる」「〜を促す」等の形で記述
- なぜその手立てが必要かを意識する

### 評価の書き方
- どの評価規準で評価するか明示（例：知①、思①）
- 評価方法を記載（ノート、発言、ワークシート等）
- 「おおむね満足できる」状態（B評価）を具体的に記述

---

## 「主体的・対話的で深い学び」の実現に向けて

### 主体的な学び
- 学習の見通しを持たせる
- 粘り強く取り組む場面を設定する
- 学習を振り返る活動を取り入れる

### 対話的な学び
- ペア・グループでの交流を設定する
- 多様な考えに触れる場面を設ける
- 先人の考え方（資料等）を活用する

### 深い学び
- 教科の「見方・考え方」を働かせる場面を設定する
- 知識を相互に関連付ける活動を取り入れる
- 問題を見いだし、解決する過程を重視する
